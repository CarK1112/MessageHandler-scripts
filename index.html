Welcome to Messagehandler-scripts Page.

The repository contains a batch of SQL script for the MessageHandler. The repository consists of 2 folders.

  • Installation_files
  • Separate_Files

Installation files

This folder is the batch of SQL scripts compressed into two files.

  • 1_MSGHANDLER_DATA.sql - Builds the table structure
  • 2_MSGHANDLER_PROC.sql - Contains the required procedures The "Separate_Files" folder contains each task separately

The installation

Connect to you MySQL Client and perform the following commands To import a database, first create a new blank database in the MySQL shell to serve as a destination for your data.

mysql > CREATE DATABASE msg;

Then log out of the MySQL shell and type the following on the command line:

mysql -uroot -p123456 msg < 1_MSGHANDLER_DATA.sql

mysql -uroot -p123456 msg < 2_MSGHANDLER_PROC.sql
